# PHP My Framework

## Overview
**PHP My Framework** is a lightweight and flexible PHP framework.

---

## Installation

### **1. Clone the Repository**
```bash
git clone https://github.com/hedeshmb/php-my-framework.git
cd php-my-framework
```

### **2. Install Dependencies**
```bash
install docker 
docker compose up --build -d
```
